# 🌱 Plant Parent Companion

## Description

Plant Parent Companion is a single-page web application that helps users manage and track their houseplants.

- 🚀 Live Preview: https://tahmina-official.github.io/SBA-316-Plant-Parent-Companion/

Users can:
- Add plants
- Track watering schedules
- Filter plants by sunlight needs
- Search plants
- Mark plants as healthy
- Remove plants
- Toggle dark mode
- Save data using localStorage
